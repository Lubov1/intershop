## Установка
1. **Клонируйте репозиторий:**
   ```sh
   git clone https://github.com/Lubov1/intershop.git
   cd intershop

2. **Соберите проект:**
   ```sh
   maven clean package

4. **Запустите команду**
    ```sh
    docker compose up

5. **Приложение будет доступно по адресу http://localhost:8080/main**
